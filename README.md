# robotics-with-MuJoCo
MuJoCo robotics project with pick-and-place manipulation, and quadruped locomotion.

We use the MoJoCo physics engine on two robots,

1. A **6-DOF robot manipulator** that does forward kinematics, inverse kinematics, PD control, and a full **pick-and-place**.
2. A **quadruped robot** that stands up, balances against a shove, and trots forward across the floor.
